Aspac Cargo

  • Project date: September 2021
  • Project URL: aspaccargo.com

Aplikasi Aspac Cargo adalah aplikasi diperuntukkan untuk mencatat dan melaporkan pengiriman barang cargo.

Aplikasi Aspac Cargo adalah aplikasi berbasis web menggunakan Laravel, sehingga mempermudah akses bagi pihak terkait dalam menginput, mengolah, dan menampilkan pelaporan data transaksi cargo.

Adapun fiturnya sebagai berikut:

  1. Mengelola data operator dan kurir
  2. Mengelola data daftar harga dan layanan
  3. Mengelola transaksi cargo beserta status transaksi
  4. Mempermudah pengguna menemukan harga dan melacak barang mereka
  5. Membuat laporan transaksi
